# MNS-veterinarni-klinika
KIV/MNS Semestrální práce 2026

Autor: Khongorzul Bayarsaikhan
Os. číslo: A24B0251P
Téma: Informační systém pro veterinární kliniku (DB1)

Popis tématu:
Cílem práce je návrh a implementace objektové aplikace pro správu malé veterinární kliniky. 
Systém bude evidovat pacienty (zvířata) a jejich majitele, kvalifikované veterináře a jejich specializace. 
Klíčovou funkcionalitou bude správa rezervací termínů, evidence lékařských ošetření a historie předepsaných léků. 
Aplikace bude pracovat s datovým modelem o 6 entitách (Zvířata, Majitelé, Veterináři, Ošetření, Léky, Rezervace), 
který umožní sledovat kompletní zdravotní kartu zvířete a plánovat kapacity kliniky.
